Controls

“1”: Display Power-2 Fractal
“2”: Display Power-4 Fractal
“3”: Display Power-6 Fractal
“4”: Display Power-8 Fractal
“5”: Display Power-16 Fractal
“6”: Toggle Shading mode
“7”: Switch view (top or side)

“z”: zoom-in
“x”: zoom-out

“w”: pan-up
“a”: pan-left
“s”: pan-down
“d”: pan-right

If no build is provided in the directory:
In the Mandelbulb directory, run the “make” command to build the project and use 
./Mandelbulb to run it.
